## Project Overview

This project researches swarm intelligence-based multi-robot cooperative SLAM technology, enabling multiple robots to simultaneously perform localization and mapping in unknown environments.

## Main Research Contents

- Distributed map fusion algorithms
- Multi-robot cooperative exploration strategies
- Swarm intelligence optimized pose graph optimization
- Real-time communication and data sharing mechanisms
