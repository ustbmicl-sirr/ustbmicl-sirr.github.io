## 题目

**Ex)** 给定椭圆区域 $4x^2 + 25y^2 = 100$，使用截面法建立体积积分（不求值），截面条件如下：

- a) 等边三角形，垂直于 x 轴
- b) 半圆，垂直于 x 轴
- c) 等腰直角三角形（斜边为底），垂直于 y 轴
- d) 正方形，垂直于 y 轴

---

## 一、什么是截面法？（通俗理解）

### 生活中的例子

**想象你在切面包**：

把一条长面包横着切成很多很多薄片，每一片的面积你都知道。那整条面包的体积是什么？就是**所有薄片面积加起来**！

```text
    面包：
    ┌─────────────────────────┐
    │ 🍞 │ 🍞 │ 🍞 │ 🍞 │ 🍞 │    ← 切成很多片
    └─────────────────────────┘

    体积 = 第1片面积×厚度 + 第2片面积×厚度 + ...
         = 所有片加起来
         = 积分！
```

### 数学语言

当薄片**无限薄**时，"加起来"就变成了**积分**：

$$V = \int_a^b A(x) \, dx$$

- $A(x)$ = 在位置 $x$ 处的截面面积（每片面包的大小）
- $dx$ = 每片的厚度（无限薄）
- $\int_a^b$ = 从 $a$ 到 $b$ 全部加起来

**一句话总结**：截面法就是**把立体切成薄片，算出每片面积，然后积分加起来**。

## 二、先搞懂这个椭圆

### 椭圆方程变形

原方程看起来很丑：$4x^2 + 25y^2 = 100$

**技巧**：两边除以100，变成标准形式

$$\frac{x^2}{25} + \frac{y^2}{4} = 1 \quad \Rightarrow \quad \frac{x^2}{5^2} + \frac{y^2}{2^2} = 1$$

**这告诉我们**：
- 椭圆沿 x 轴伸展到 $\pm 5$（横着长）
- 椭圆沿 y 轴伸展到 $\pm 2$（竖着短）

### 椭圆长什么样？

**形状特点**：
- 横轴方向：从 $x = -5$ 到 $x = 5$（长轴 = 10）
- 纵轴方向：从 $y = -2$ 到 $y = 2$（短轴 = 4）
- 这是一个**横向拉长**的椭圆（像被压扁的圆）

**示意图**（建议参考原题图片中的椭圆）：

```text
          y
          ↑
        2 ┼·············(0,2)·············
          │         .           .
          │      .                 .
          │    .                     .
          │  .                         .
    ──────┼─.───────────────────────────.──→ x
    (-5,0)│  .                         .  (5,0)
          │    .                     .
          │      .                 .
          │         .           .
       -2 ┼·············(0,-2)············
          │
```

> 提示：椭圆就像一个被从上下压扁的圆，或者想象把一个圆横向拉长。

### 最重要的公式：弦长

**什么是弦长？** 就是椭圆上某一点"横着"或"竖着"能切多宽。

**垂直于 x 轴切**（竖着切）：

在位置 $x$ 处，椭圆的高度是多少？

从椭圆方程解出 $y$：
$$y = \pm\frac{2}{5}\sqrt{25 - x^2}$$

上面到下面的距离（弦长）：
$$\text{弦长} = 2y = \frac{4}{5}\sqrt{25 - x^2}$$

**垂直于 y 轴切**（横着切）：

在位置 $y$ 处，椭圆的宽度是多少？

$$x = \pm\frac{5}{2}\sqrt{4 - y^2}$$

左边到右边的距离（弦长）：
$$\text{弦长} = 2x = 5\sqrt{4 - y^2}$$

## 三、四道题逐个击破

### 题目 a：等边三角形截面（垂直于 x 轴）

#### 想象一下

你拿一把刀，**垂直于 x 轴**去切这个立体。每切一刀，截面都是一个**等边三角形**。

三角形的底边 = 椭圆在该位置的弦长

```text
    侧面看：                    截面（等边三角形）：

         ╱╲                           ╱╲
        ╱  ╲                         ╱  ╲
       ╱    ╲                       ╱    ╲  ← 三边相等
      ╱──────╲                     ╱──────╲
     ╱        ╲                    底边 = 弦长
    ╱──────────╲
    ─────────────→ x
   -5     0     5
```

#### 三步求解

**第一步：底边是多少？**

底边 = 椭圆在 $x$ 处的弦长 = $\frac{4}{5}\sqrt{25-x^2}$

**第二步：等边三角形面积公式**

边长为 $s$ 的等边三角形面积 = $\frac{\sqrt{3}}{4}s^2$

（这个公式要背！）

**第三步：代入得到 A(x)**

$$A(x) = \frac{\sqrt{3}}{4} \times \left(\frac{4}{5}\sqrt{25-x^2}\right)^2 = \frac{4\sqrt{3}}{25}(25-x^2)$$

#### 最终答案

$$\boxed{V = \int_{-5}^{5} \frac{4\sqrt{3}}{25}(25-x^2) \, dx}$$

### 题目 b：半圆截面（垂直于 x 轴）

#### 想象一下

每切一刀，截面是一个**半圆**。半圆的直径 = 椭圆的弦长

```text
    侧面看：                    截面（半圆）：

        ╭───╮                      ╭─────╮
       ╱     ╲                    ╱       ╲
      ╭───────╮                  │         │
     ╱         ╲                 ╰─────────╯
    ╭───────────╮                直径 = 弦长
    ─────────────→ x
   -5     0     5
```

#### 三步求解

**第一步：直径和半径**

直径 = 弦长 = $\frac{4}{5}\sqrt{25-x^2}$

半径 = 直径 ÷ 2 = $\frac{2}{5}\sqrt{25-x^2}$

**第二步：半圆面积公式**

半圆面积 = $\frac{1}{2}\pi r^2$

**第三步：代入得到 A(x)**

$$A(x) = \frac{1}{2}\pi \times \left(\frac{2}{5}\sqrt{25-x^2}\right)^2 = \frac{2\pi}{25}(25-x^2)$$

#### 最终答案

$$\boxed{V = \int_{-5}^{5} \frac{2\pi}{25}(25-x^2) \, dx}$$

---

### 题目 c：等腰直角三角形（斜边为底，垂直于 y 轴）

#### 想象一下

这次是**垂直于 y 轴**切！截面是等腰直角三角形，而且**斜边是底边**。

```text
    侧面看：                    截面（等腰直角三角形）：

                                      ╱│
        ╱╲                           ╱ │
       ╱  ╲                         ╱  │ ← 两腰相等
      ╱────╲                       ╱   │    且夹90°
    ──────────→ y                 ╱────┘
     -2  0  2                     斜边（底）= 弦长
```

#### 小知识：等腰直角三角形

如果斜边 = $h$，那么：
- 两条腰各为 $\frac{h}{\sqrt{2}}$
- 面积 = $\frac{h^2}{4}$（记住这个！）

#### 三步求解

**第一步：斜边是多少？**

垂直于 y 轴，斜边 = 椭圆在 $y$ 处的弦长 = $5\sqrt{4-y^2}$

**第二步：面积公式**

斜边为 $h$ 的等腰直角三角形面积 = $\frac{h^2}{4}$

**第三步：代入得到 A(y)**

$$A(y) = \frac{1}{4} \times \left(5\sqrt{4-y^2}\right)^2 = \frac{25}{4}(4-y^2)$$

#### 最终答案

注意：这次积分变量是 $y$，范围是 $-2$ 到 $2$

$$\boxed{V = \int_{-2}^{2} \frac{25}{4}(4-y^2) \, dy}$$

---

### 题目 d：正方形截面（垂直于 y 轴）

#### 想象一下

最简单的一个！截面是正方形，边长 = 椭圆的弦长

```text
    侧面看：                    截面（正方形）：

       ┌────┐                   ┌────────┐
       │    │                   │        │
       │    │                   │        │ ← 四边相等
       │    │                   │        │
       └────┘                   └────────┘
    ──────────→ y               边长 = 弦长
     -2  0  2
```

#### 三步求解

**第一步：边长是多少？**

边长 = 椭圆在 $y$ 处的弦长 = $5\sqrt{4-y^2}$

**第二步：正方形面积**

面积 = 边长² = $s^2$

**第三步：代入得到 A(y)**

$$A(y) = \left(5\sqrt{4-y^2}\right)^2 = 25(4-y^2)$$

#### 最终答案

$$\boxed{V = \int_{-2}^{2} 25(4-y^2) \, dy}$$

## 四、总结：做题套路

### 万能三步法

```text
第一步：找截面的"边"
        ↓
        边 = 椭圆的弦长
        ↓
第二步：用面积公式算 A
        ↓
        代入弦长表达式
        ↓
第三步：写积分
        ↓
        确定积分变量和范围
```

### 必背公式卡片

| 形状 | 已知量 | 面积公式 |
| :---: | :---: | :---: |
| 正方形 | 边长 $s$ | $s^2$ |
| 圆 | 半径 $r$ | $\pi r^2$ |
| 半圆 | 半径 $r$ | $\frac{\pi r^2}{2}$ |
| 等边三角形 | 边长 $s$ | $\frac{\sqrt{3}}{4}s^2$ |
| 等腰直角三角形 | 腰长 $a$ | $\frac{a^2}{2}$ |
| 等腰直角三角形 | 斜边 $h$ | $\frac{h^2}{4}$ |

### 本题答案汇总

| 题号 | 截面 | 积分方向 | 体积积分 |
| :---: | :---: | :---: | :---: |

## 五、常见问题

**Q1：怎么判断用 x 还是 y 积分？**

> 看题目说"垂直于哪个轴"。垂直于 x 轴就对 x 积分，垂直于 y 轴就对 y 积分。

**Q2：积分上下限怎么确定？**

> 看椭圆在那个方向上的范围。x 方向是 -5 到 5，y 方向是 -2 到 2。

**Q3：为什么要先化简椭圆方程？**

> 化简后能直接看出椭圆的"长"和"宽"，方便确定积分限和计算弦长。

**Q4：截面积公式记不住怎么办？**

> 常用的就那几个，做几道题就记住了。考试可以带公式表的话就不用背。
